## 6.1 Docker 服务命令

1. 启动 Docker 服务

```shell
$ systemctl start docker
```

2. 停止 Docker 服务

```shell
$ systemctl stop docker
```

3. 重启 Docker 服务

```shell
$ systemctl restart docker
```

4. 设置开机启动 Docker 服务

```shell
$ systemctl enable docker
```

5. 查看 Docker 服务状态

```shell
$ systemctl status docker
```
